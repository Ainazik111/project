# Kyrgyzstan Atlas — Nomad's Atlas

Интерактивный путеводитель по Кыргызстану. Сайт знакомит с историей, культурой, кухней, природой и достопримечательностями страны.

## Возможности

- **Главная** — информация о стране, население, язык, национальные игры
- **Бишкек** — intro + каталог мест (парки, музеи, базары, рестораны)
- **История** — эпохи, ключевые события, культура (с аудио и видео)
- **Кухня** — национальные блюда и традиционные напитки
- **Природа** — статьи о природе с фото и описанием
- **Достопримечательности** — места по всему Кыргызстану
- **Гид** — полезные приложения, советы, экстренные контакты, Instagram
- **Перевод** — переключение между русским и английским языком

## Стек

- Python 3.10+ / Django 6.0
- SQLite (разработка) / PostgreSQL (продакшн)
- django-modeltranslation — перевод контента в БД
- WhiteNoise — статика в продакшне
- Gunicorn — WSGI-сервер
- Pillow — обработка изображений

## Установка

```bash
git clone https://github.com/Ainazik111/project.git
cd project
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Сайт будет доступен по адресу `http://127.0.0.1:8000`.

## Использование

- **Русский**: `http://127.0.0.1:8000/` (язык по умолчанию)
- **English**: `http://127.0.0.1:8000/en/`
- **Админка**: `http://127.0.0.1:8000/admin/`
  - Логин: `admin`
  - Пароль: `admin`

Для смены языка используйте переключатель в правом верхнем углу навигационной панели.

## Перевод

Система перевода включает два механизма:

1. **Шаблоны** — строки в `{% trans %}` переведены в `locale/en/LC_MESSAGES/django.po`
2. **База данных** — модель `django-modeltranslation` добавляет поля `_ru` и `_en` к каждой модели. Переводы заполняются через админку.

Чтобы добавить новый перевод:

```bash
python manage.py makemessages -l en
# отредактировать locale/en/LC_MESSAGES/django.po
python manage.py compilemessages
```

## Продакшн

Проект готов к деплою на Heroku:

```bash
heroku create
git push heroku main
heroku run python manage.py migrate
```

Переменные окружения:

- `SECRET_KEY` — секретный ключ Django
- `DEBUG` — `True` / `False`
- `ALLOWED_HOSTS` — список хостов через запятую
